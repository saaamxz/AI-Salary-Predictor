# AI Salary Predictor 

تطبيق ويب ذكي يقوم بتوقع مرتبات المبرمجين بناءً على بياناتهم الشخصية والمهنية باستخدام تقنيات التعلم الآلي (Machine Learning).

##  Project Goal
الهدف هو تحليل العلاقة بين سنوات الخبرة والتقييم الشخصي للمبرمج وبين متوسط الراتب المتوقع، وتوفير أداة بسيطة للمستخدم للحصول على تقدير سريع لراتبه.

##  Technical Stack
- **Languages:** Python
- **Libraries:** Pandas, NumPy (Data Processing), Scikit-Learn (Model Training), Streamlit (Web Framework).
- **Core Concept:** Linear Regression (الانحدار الخطي).

## Key Features
- **Data-Driven Predictions:** النموذج يعتمد على بيانات تاريخية لتوقع الأرقام.
- **User-Friendly Interface:** واجهة بسيطة بـ Streamlit لإدخال البيانات والحصول على النتيجة فوراً.
- **Real-time Processing:** معالجة فورية للمدخلات وإظهار النتيجة.
